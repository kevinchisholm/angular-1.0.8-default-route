## Dependencies

What you need to run this app:

* [Node.Js](https://nodejs.org)

## Instructions

### Setup

## 1 - Clone this repo: 

```
git clone https://github.com/kevinchisholm/angular-1.0.8-default-route.git
```

## 2 - Install Dependencies:

```
npm install
```

## 3 - Start the Web Server:

```
npm start
```
